# TikTok AI Automation Workspace

This project is an AI-powered automation system for TikTok US e-commerce.

## Features
- Product trend scraping
- Viral score ranking
- AI video script generation (UGC style 10–15s)
- Hashtag & title optimization
- Automated content pipeline

## Structure
- scraper/       -> product data collection
- scripts/       -> AI script generation
- video/         -> video pipeline
- workflow/      -> automation logic
- api/           -> external integrations

## Goal
Build a fully automated TikTok content + product system using AI + GitHub Actions + Paperclip.
